一、WinSCP

概念介绍

​ WinSCP是一个Windows环境下使用SSH的开源图形化SFTP客户端。同时支持SCP协议。它的主要功能就是在本地与远程计算机间安全的复制文件。.winscp也可以链接其他系统,比如linux系统

使用方法

​ 客户端装在本地，WinSCP运行脚本也放到本地运行，这个本地可以是正式端也可以是测试端，只不过脚本中的同步指令会因地制宜，建议装在正式端。下载链接：https://winscp.net/eng/download.php

1.图形化界面

​ Hostname写远程（测试端）IP，port写22，username和password自己知道的

2.命令提示符

​ 1.cd到WinSCP.exe的目录下，输入winscp进入winscp操作介面
​ 2.连接到远程机器：open sftp://用户名:密码@远程ip:22
​ 3.synchronize remote 本地文件夹路径（正式端） /远端文件夹路径（测试端）

例:synchronize remote C:\Users\admin\Desktop\test /C:\Users\Administrator\Desktop\test2
本地的test文件夹内容就被remote到远端电脑文件夹test2中了，但是我需要实现，删除本地文件，远端文件也同步删除，则需要输入option synchdelete on

粗略解释一下WinSCP指令吧，可通过help或者help command(具体指令)查看提示

二、OpenSSH

基本概念

【概念】Secure Shell缩写是SSH， 由IETF的网络工作小组（Network Working Group）所制定，SSH是一项创建在应用层和传输层基础上的安全协议，为计算机的shell提供安全的传输和使用环境传统的网络服务程序，如rsh FTP POP Telnet本质上都是不安全的，因为它们在网络 上是用明文传送数据，用户账号，和用户口令，很容易受到中间人攻击方式的攻击，存在另一个人或者一台机器冒充真正的服务器接收用户传给服务器的数据，然后在冒充用户把数据传给真正的服务器，而SSH是目前比较可靠，专为远程登录回话和其他网络服务提供安全性的协议，利用SSH协议可以有效的防止远程管理过程中的信息泄露问题。通过SSH可以对所有的输出数据进行加密，也能够防止DNS欺骗和IP欺骗SSH的另一个优点是传输的数据可以是经过压缩的，可以加快传输的速度，SSH还有很多功能，可以代替Telnet， 可以作为FTP， POP，甚至为PPP提供一个安全的通道

OpenSSH是SSH的开源实现，因此用户可以免费使用到这种安全服务。
详见：https://www.jianshu.com/p/f7f1d6f16d21

使用方法

  1. 访问https://github.com/PowerShell/Win32-OpenSSH/releases，下载PC对应的二进制包，如OpenSSH-Win64.zip，并解压到文件接收端（测试端）；

  2. 在C:\Program Files文件夹中新建OpenSSH文件夹，并将（1）中解压的所有文件copy到OpenSSH文件夹中，此处都需要Administrator的权限；

  3. 管理员权限运行command，并切换到C:\ProgramFiles\OpenSSH这个文件夹，运行以下命令：powershell.exe -ExecutionPolicy Bypass -File install-sshd.ps1

  4. 设置防火墙：Control Panel > System and Security > Windows Firewall >Advanced Settings > Inbound Rules ，右键新建rule，选择port，tcp，固定端口22，Name写sshd，description写OpenSSH Server (sshd)，完成。

  5. 开启防火墙：Control Panel > System and Security > Administrative Tools，打开Services应用，找到刚刚的sshd，右键属性，startup type选automatic自启动，最后点击Start Service。
